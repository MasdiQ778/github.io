Apa Itu IP Statis? Penjelasan Lengkap

IP Statis adalah alamat IP yang tidak berubah dan tetap digunakan oleh perangkat dalam jaringan. IP ini dikonfigurasi secara manual dan tetap sama meskipun perangkat di-restart atau jaringan berubah.

ğŸ”¹ Perbedaan IP Statis dan Dinamis

  • IP Statis: Tetap dan diatur secara manual.
  • IP Dinamis: Berubah-ubah, ditetapkan otomatis oleh DHCP server.

ğŸ”¹ Kelebihan IP Statis

  • Mudah diakses dari luar jaringan (misalnya untuk server website atau CCTV).
  • Koneksi lebih stabil dan bisa diandalkan.
  • Mudah dikelola dalam jaringan besar.

ğŸ”¹ Kekurangan IP Statis

  • Perlu konfigurasi manual.
  • Lebih mudah dilacak oleh pihak luar.
  • Biasanya dikenakan biaya tambahan oleh ISP.

Jika kamu ingin menjalankan server sendiri di rumah atau kantor, IP statis adalah pilihan yang ideal karena tidak berubah dan mudah diakses dari mana saja.
